[q1-2017] Wireless Control of UR5 Manipulator

Semester project, Aalborg University.

Summary: A wireless control system for an industrial manipulator using an iPad as the interface. This project entailed the implementation of a Kalman filter to capture the attitude motion of the iPad, ensuring smooth and precise control, and a velocity control system for the manipulator, allowing for seamless manipulation of the industrial arm based on the input from the iPad. Furthermore, I successfully established a wireless interface between the iPad and the manipulator, enabling real-time communication and control.

Keywords: Parallel platform, inverse kinematics, control, matlab, arduino.
